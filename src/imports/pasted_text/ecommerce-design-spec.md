# SYSTEM PROMPT: MODERN FASHION E-COMMERCE PLATFORM (BLACK & WHITE MINIMALISM)

Design and generate a complete fashion e-commerce website with a premium, modern, minimal black & white aesthetic. The experience must feel high-end, smooth, and fully interactive.

## CORE BRAND STYLE
- Primary colors: Black (#000000) & White (#FFFFFF)
- Accent: subtle grayscale, soft glow, or thin neon lines (optional)
- Style: Minimalist, luxury fashion brand (inspired by high-end brands)
- Typography: Clean, bold headings + elegant body text
- UI feeling: Smooth, modern, high conversion focused

---

## MAIN PAGES & STRUCTURE

### 1. Homepage
- Hero banner (fashion campaign style)
- Featured products section
- Flash sale highlight (countdown timer)
- Reward / loyalty introduction
- Blog preview section
- Call-to-action buttons (Shop Now, Explore)

---

### 2. Product Listing Page
- Grid layout of products
- Filter (category, price, size, color)
- Sort (newest, best seller, price)
- Hover effect (image swap or quick preview)
- Each product links to detail page

---

### 3. Product Detail Page
- Large product images gallery
- Product info (name, price, description)
- Size selector, quantity selector
- Add to cart button (fully interactive)
- Related products
- Reviews / ratings section

---

### 4. Cart & Checkout
- Add / remove items
- Update quantity
- Show total price
- Apply discount / reward points
- Checkout flow (shipping info, payment)
- Order confirmation page

---

### 5. Reward & Loyalty System
- User earns points when purchasing
- Points can be redeemed for discounts
- Display user points dashboard
- Reward exchange page

---

### 6. Flash Sale System
- Limited-time deals
- Countdown timer
- Highlight urgency (low stock, ending soon)
- Dedicated flash sale page + detail page

---

### 7. Blog / Article System
- Blog listing page
- Blog detail page
- Categories (fashion tips, trends, news)
- Clean reading layout

---

### 8. User Account
- Profile page
- Order history
- Saved products (wishlist)
- Reward points tracking

---

## ADMIN DASHBOARD (FULL CRUD SYSTEM)

Admin can manage ALL system data:
- Products (Create, Read, Update, Delete)
- Categories
- Orders
- Users
- Blog posts
- Flash sale campaigns
- Reward system rules

Dashboard UI:
- Clean, data-driven interface
- Tables, charts, analytics
- Easy navigation sidebar

---

## INTERACTION REQUIREMENTS
- All buttons must be clickable and functional
- Smooth transitions & animations
- Hover effects for interactive elements
- Loading states, empty states

- Micro-interactions (hover, click feedback)

- Premium product showcase style
